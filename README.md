# Proyecto de Sitio Web de Recetas — Ejemplo de Buenas Prácticas SEO Técnico

Este proyecto fue desarrollado como un **caso de estudio educativo** para comprender y aplicar las **buenas prácticas de SEO técnico** en un sitio web sencillo.  
A través de un ejemplo realista (Una página web de recetas caseras) se busca ilustrar cómo la correcta estructura del código, el uso de metadatos, la optimización de recursos y la accesibilidad influyen en el rendimiento y la visibilidad de un proyecto web.

## Objetivo del Proyecto
El propósito principal de este proyecto es **aprender a identificar, corregir y aplicar las buenas prácticas de SEO técnico** que mejoran el posicionamiento en motores de búsqueda, incluyendo:
- Estructura semántica del HTML.  
- Optimización de etiquetas `<meta>`, Open Graph y Twitter Cards.  
- Implementación de datos estructurados con **JSON-LD**.  
- Configuración de archivos esenciales: `robots.txt` y `sitemap.xml`.  
- Estrategias para mejorar la velocidad de carga y la indexabilidad.
