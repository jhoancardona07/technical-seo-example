# SEO Recipes — Proyecto de ejemplo para evidencias

- `robots.txt` en raíz (User-agent, Disallow/Allow, Sitemaps).
- `sitemap.xml` + `sitemap_index.xml` en raíz (URLs limpias y https).
- `<meta name="robots">` en todas las páginas; `noindex` en privacidad/terminos/resultados.
- X-Robots-Tag ejemplos para Apache/Netlify/Nginx/Express, con archivos de prueba.
- OG/Twitter, canónicos, hreflang, títulos/description únicos.
