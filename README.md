# SEO Recipes — Proyecto de ejemplo para evidencias

- `robots.txt` en la raíz (User-agent, Disallow/Allow, Sitemaps).
- `sitemap.xml` + `sitemap_index.xml` en la raíz (URLs limpias y https).
- `<meta name="robots">` en todas las páginas; `noindex` en las páginas de privacidad/terminos/resultados.
- X-Robots-Tag: Un par de ejemplos para Apache/Netlify/Nginx/Express, con archivos de prueba.
- OG/Twitter, canónicos, hreflang, con títulos y description únicos.
